# 第四章：表达调控

## 4.1 基因表达调控概述

### 调控层次

<concept id="gene_regulation">基因表达调控</concept>发生在多个层次：

```
DNA → 转录 → RNA加工 → 翻译 → 蛋白质修饰
 ↑      ↑       ↑        ↑        ↑
染色质  转录   转录后   翻译    翻译后
```

| 层次 | 调控点 | 特点 |
|------|--------|------|
| <entity type="概念">染色质水平</entity> | DNA可及性 | 表观遗传，长期调控 |
| <entity type="概念">转录水平</entity> | 转录起始 | 最主要调控点 |
| <entity type="概念">转录后水平</entity> | RNA加工、稳定性 | 快速响应 |
| <entity type="概念">翻译水平</entity> | 翻译起始 | 应急调控 |
| <entity type="概念">翻译后水平</entity> | 蛋白质修饰、定位 | 功能激活 |

## 4.2 原核生物基因调控

### 操纵子模型

<concept id="operon">操纵子</concept>（Operon）是原核生物基因调控的基本单位：

```
启动子 - 操纵基因 - 结构基因群
   ↑         ↑
RNA聚合酶  阻遏蛋白
```

| 元件 | 功能 |
|------|------|
| <entity type="概念">启动子</entity> | RNA聚合酶结合位点 |
| <entity type="概念">操纵基因</entity>（Operator） | 阻遏蛋白结合位点 |
| <entity type="概念">结构基因</entity> | 编码功能蛋白 |
| <entity type="概念">调节基因</entity> | 编码调控蛋白 |

### 乳糖操纵子

<entity type="概念">乳糖操纵子</entity>（Lac operon）：

**负调控**（阻遏）：
- 无乳糖时：<entity type="蛋白">阻遏蛋白</entity>结合操纵基因，转录关闭
- 有乳糖时：乳糖作为<entity type="分子">诱导物</entity>，阻遏蛋白失活，转录开启

**正调控**（激活）：
- 葡萄糖缺乏时：<entity type="分子">cAMP</entity>升高，<entity type="蛋白">CAP蛋白</entity>激活转录

### 色氨酸操纵子

<entity type="概念">色氨酸操纵子</entity>（Trp operon）：

**阻遏型操纵子**：
- 无色氨酸时：转录开启
- 有色氨酸时：色氨酸作为<entity type="分子">辅阻遏物</entity>，阻遏蛋白激活，转录关闭

**衰减作用**（Attenuation）：
- 前导肽的翻译影响转录终止
- 转录-翻译偶联的精细调控

## 4.3 真核生物转录调控

### 顺式作用元件

<concept id="cis_element">顺式作用元件</concept>：DNA上的调控序列

| 元件 | 位置 | 功能 |
|------|------|------|
| <entity type="概念">启动子</entity> | 转录起始点附近 | 基础转录装置组装 |
| <entity type="概念">增强子</entity>（Enhancer） | 远端（1kb-1Mb） | 显著增强转录效率 |
| <entity type="概念">沉默子</entity>（Silencer） | 远端 | 抑制转录 |
| <entity type="概念">绝缘子</entity>（Insulator） | 边界 | 阻断增强子/沉默子作用 |

### 反式作用因子

<concept id="trans_factor">反式作用因子</concept>：调控蛋白

**转录因子结构**：
- <entity type="结构">DNA结合域</entity>：识别特定序列
- <entity type="结构">转录激活域</entity>：招募转录 machinery
- <entity type="结构">二聚化结构域</entity>：形成同源/异源二聚体

**常见DNA结合域**：
- <entity type="结构">同源异型域</entity>（Homeodomain）
- <entity type="结构">锌指结构</entity>（Zinc finger）
- <entity type="结构">碱性亮氨酸拉链</entity>（bZIP）
- <entity type="结构">螺旋-环-螺旋</entity>（HLH）

### 转录调控网络

**组合调控**：
- 多个转录因子协同作用
- 产生特异性的时空表达模式

**实例**：<entity type="概念">MyoD</entity>与肌肉分化
- <entity type="蛋白">MyoD</entity>是<entity type="概念">碱性HLH</entity>转录因子
- 激活肌肉特异性基因表达
- 可诱导成纤维细胞转化为肌细胞

## 4.4 转录后调控

### RNA加工调控

**选择性剪接**（Alternative splicing）：
- 一个基因产生多种mRNA异构体
- <entity type="概念">外显子</entity>跳跃、<entity type="概念">内含子</entity>保留
- 人类约95%的基因存在选择性剪接

**实例**：<entity type="基因">α-原肌球蛋白</entity>基因
- 14个外显子，通过选择性剪接产生多种组织特异性蛋白

### RNA稳定性调控

**mRNA降解途径**：
- <entity type="概念">脱腺苷酸化</entity>：去除poly(A)尾
- <entity type="概念">脱帽</entity>：去除5'帽
- <entity type="概念">外切酶降解</entity>：5'→3'或3'→5'

**调控元件**：
- <entity type="概念">AU富含元件</entity>（ARE）：促进降解
- <entity type="概念">miRNA</entity>：通过<entity type="概念">RNA干扰</entity>抑制翻译或促进降解

### miRNA调控

<entity type="分子">miRNA</entity>（微小RNA）：
- 长度：21-25 nt
- 来源：RNA聚合酶II转录，经<entity type="酶">Drosha</entity>和<entity type="酶">Dicer</entity>加工
- 机制：与<entity type="复合物">RISC复合物</entity>结合，靶向mRNA

**调控方式**：
- 完全互补：mRNA切割
- 部分互补：翻译抑制

## 4.5 翻译调控

### 翻译起始调控

**eIF4E结合蛋白**（<entity type="蛋白">4E-BP</entity>）：
- 与<entity type="因子">eIF4E</entity>竞争结合
- <entity type="激酶">mTOR激酶</entity>磷酸化4E-BP，释放eIF4E
- 营养信号调控翻译

**铁响应元件**（<entity type="概念">IRE</entity>）：
- <entity type="蛋白">铁调节蛋白</entity>（IRP）结合IRE
- 铁缺乏时：抑制<entity type="蛋白">铁蛋白</entity>mRNA翻译
- 铁充足时：IRP失活，翻译进行

### 翻译延伸与终止调控

**罕见密码子**：
- 对应tRNA丰度低
- 减慢翻译速度，影响蛋白质折叠

**通读**（Read-through）：
- 终止密码子被识别为编码氨基酸
- 产生延伸的蛋白质

## 4.6 表观遗传调控

### DNA甲基化

<entity type="修饰">DNA甲基化</entity>：
- 位点：<entity type="碱基">CpG岛</entity>中的<entity type="碱基">胞嘧啶</entity>
- 酶：<entity type="酶">DNA甲基转移酶</entity>（DNMT）

**生物学意义**：

### 组蛋白修饰

<concept id="histone_modification">组蛋白修饰</concept>：

| 修饰 | 位点 | 效应 |
|------|------|------|
| <entity type="修饰">乙酰化</entity>（Ac） | 赖氨酸 | 激活转录 |

**组蛋白密码**：不同修饰组合产生特定调控效应

<entity type="复合物">染色质重塑复合物</entity>：
- <entity type="复合物">SWI/SNF</entity>：利用ATP水解能量重塑核小体

## 本章小结

- 基因表达调控发生在多个层次，转录水平最重要
- 原核生物以操纵子为单位进行负/正调控
- 真核生物通过顺式元件和反式因子精细调控
- 转录后调控包括剪接、稳定性、miRNA等
- 表观遗传调控不改变DNA序列，但影响基因表达
