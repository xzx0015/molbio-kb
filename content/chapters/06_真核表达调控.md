# 第六章：真核生物基因表达调控

## 6.1 真核基因调控概述

### 与原核生物的差异

| 特征 | 原核生物 | 真核生物 |
|------|---------|---------|
| 转录翻译 | 偶联 | 分隔（核膜） |
| 基因组织 | 操纵子 | 单基因 |
| 染色质 | 无 | 有核小体 |
| 主要调控点 | 转录 | 多水平 |
| RNA加工 | 无 | 复杂（加帽、加尾、剪接） |

### 调控层次

<concept id="eukaryotic_regulation">真核基因表达调控</concept>发生在多个层次：

```
染色质水平 → 转录水平 → RNA加工 → mRNA运输 → 翻译水平 → 蛋白质修饰
    ↑           ↑          ↑          ↑          ↑          ↑
表观遗传   转录因子   选择性剪接   核质运输   翻译起始   信号转导
```

## 6.2 染色质水平调控

### DNA甲基化

<entity type="修饰">DNA甲基化</entity>：
- **位点**：<entity type="碱基">CpG岛</entity>中的胞嘧啶（C→5mC）
- **酶**：<entity type="酶">DNA甲基转移酶</entity>（DNMT1维持，DNMT3a/3b从头）
- **效应**：通常抑制基因表达

**生物学意义**：
- <entity type="概念">基因组印记</entity>（Imprinting）：亲本特异性表达
- <entity type="概念">X染色体失活</entity>：雌性X染色体剂量补偿
- 转座子沉默
- 肿瘤抑制基因沉默

### 组蛋白修饰

<concept id="histone_modification">组蛋白修饰</concept>：

| 修饰 | 位点 | 效应 |
|------|------|------|
| <entity type="修饰">乙酰化</entity>（Ac） | 赖氨酸 | 激活转录 |
| <entity type="修饰">甲基化</entity>（Me） | 赖氨酸/精氨酸 | H3K4me3激活，H3K27me3抑制 |
| <entity type="修饰">磷酸化</entity>（P） | 丝氨酸 | 调控染色质结构 |
| <entity type="修饰">泛素化</entity>（Ub） | 赖氨酸 | 调控转录 |

**组蛋白密码假说**：
- 不同修饰组合产生特定调控效应
- 修饰之间相互影响（交叉对话）

**功能**：

## 6.3 转录水平调控

### 顺式作用元件

<concept id="cis_element">顺式作用元件</concept>：DNA上的调控序列

| 元件 | 位置 | 功能 |
|------|------|------|
| <entity type="概念">启动子</entity>（Promoter） | 转录起始点附近 | 基础转录装置组装 |
| <entity type="概念">增强子</entity>（Enhancer） | 远端（1kb-1Mb） | 显著增强转录效率 |
| <entity type="概念">沉默子</entity>（Silencer） | 远端 | 抑制转录 |
| <entity type="概念">绝缘子</entity>（Insulator） | 边界 | 阻断增强子/沉默子作用 |

**增强子特征**：
- 方向无关性
- 位置灵活性（上游、下游、内含子）
- 组织特异性
- 与启动子形成染色质环

### 反式作用因子

<concept id="trans_factor">反式作用因子</concept>：调控蛋白

**转录因子结构**：
- <entity type="结构">DNA结合域</entity>：识别特定序列
- <entity type="结构">转录激活域</entity>：招募转录 machinery
- <entity type="结构">二聚化结构域</entity>：形成同源/异源二聚体
- <entity type="结构">核定位信号</entity>（NLS）：进入细胞核

**常见DNA结合域**：
| 结构域 | 特征 | 代表因子 |
|--------|------|---------|
| <entity type="结构">同源异型域</entity>（Homeodomain） | 螺旋-转角-螺旋变体 | HOX蛋白 |
| <entity type="结构">锌指</entity>（Zinc finger） | Zn²⁺稳定 | SP1、TFIIIA |
| <entity type="结构">碱性亮氨酸拉链</entity>（bZIP） | 亮氨酸拉链二聚化 | AP-1、CREB |
| <entity type="结构">螺旋-环-螺旋</entity>（HLH） | HLH二聚化 | MyoD、E47 |

### 转录调控网络

**组合调控**：
- 多个转录因子协同作用
- 产生特异性的时空表达模式
- 增强调控特异性

**实例**：
- <entity type="蛋白">MyoD</entity>与肌肉分化
- <entity type="蛋白">Oct4</entity>、<entity type="蛋白">Sox2</entity>、<entity type="蛋白">Nanog</entity>与干细胞多能性

## 6.4 转录后调控

### 选择性剪接

<concept id="alternative_splicing">选择性剪接</concept>：
- 一个基因产生多种mRNA异构体
- 人类约95%的基因存在选择性剪接
- 增加蛋白质组多样性

**剪接模式**：
| 模式 | 说明 |
|------|------|
| 外显子跳跃 | 跳过某个外显子 |
| 互斥外显子 | 两个外显子只保留一个 |
| 内含子保留 | 保留某个内含子 |
| 可变5'剪接位点 | 使用不同的5'剪接位点 |
| 可变3'剪接位点 | 使用不同的3'剪接位点 |

**调控因子**：
- <entity type="蛋白">SR蛋白</entity>：促进剪接
- <entity type="蛋白">hnRNP蛋白</组织>：抑制剪接

### RNA编辑

<concept id="rna_editing">RNA编辑</concept>：
- 转录后改变RNA序列
- 脱氨酶（ADAR、APOBEC）介导
- 实例：APOB mRNA的C→U编辑

### RNA稳定性调控

**mRNA降解途径**：
1. <entity type="概念">脱腺苷酸化</entity>：去除poly(A)尾
2. <entity type="概念">脱帽</entity>：去除5'帽
3. <entity type="概念">外切酶降解</entity>：5'→3'或3'→5'

**调控元件**：
- <entity type="概念">AU富含元件</entity>（ARE）：促进降解
- <entity type="概念">miRNA</entity>：通过<entity type="概念">RNA干扰</entity>抑制翻译或促进降解

### miRNA调控

<entity type="分子">miRNA</entity>（微小RNA）：
- 长度：21-25 nt
- 来源：RNA聚合酶II转录，经<entity type="酶">Drosha</entity>和<entity type="酶">Dicer</entity>加工
- 机制：与<entity type="复合物">RISC复合物</entity>结合，靶向mRNA

**调控方式**：
- 完全互补：mRNA切割
- 部分互补：翻译抑制

## 6.5 翻译水平调控

### 全局翻译调控

**eIF2磷酸化**：
- 应激条件下eIF2α磷酸化
- 抑制全局翻译起始
- 但激活特定应激基因翻译

**mTOR通路**：
- <entity type="激酶">mTOR激酶</entity>感知营养信号
- 磷酸化<entity type="蛋白">4E-BP</entity>，释放eIF4E
- 促进翻译起始

### 特异性翻译调控

**铁响应元件**（<entity type="概念">IRE</entity>）：
- <entity type="蛋白">铁调节蛋白</entity>（IRP）结合IRE
- 铁缺乏时：抑制<entity type="蛋白">铁蛋白</entity>mRNA翻译
- 铁充足时：IRP失活，翻译进行

**上游开放阅读框**（uORF）：
- 位于主ORF上游
- 抑制主ORF翻译
- 应激条件下被绕过

## 6.6 信号转导与基因表达

### 信号通路

**MAPK通路**：
- 生长因子 → Ras → Raf → MEK → ERK
- ERK入核激活转录因子

**Wnt/β-catenin通路**：
- Wnt信号稳定β-catenin
- β-catenin入核激活TCF转录因子

**TGF-β/Smad通路**：
- TGF-β受体磷酸化Smad2/3
- Smad复合物入核调控转录

### 激素调控

**类固醇激素**：
- 脂溶性，可穿过细胞膜
- 与核受体结合
- 激素-受体复合物直接调控转录

**实例**：
- 雌激素受体（ER）
- 糖皮质激素受体（GR）

## 本章小结

- 真核基因调控发生在多水平，比原核更复杂
- 染色质修饰（DNA甲基化、组蛋白修饰）是表观遗传调控的基础
- 转录因子通过顺式元件和反式因子精细调控
- 转录后调控（选择性剪接、miRNA）增加调控层次
- 信号转导通路将胞外信号转化为基因表达变化
