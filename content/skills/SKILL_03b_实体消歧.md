# SKILL_03b_实体消歧.md

## 同名基因/蛋白跨物种消歧策略

---

## 一、消歧概述

### 1.1 为什么需要消歧？

分子生物学中存在大量同名异义现象：
- **跨物种同名**: 人类p53 vs 小鼠p53 vs 大鼠p53
- **基因家族**: p53家族（p53, p63, p73）
- **亚型差异**: 同一基因的不同剪接变体
- **历史命名**: 同一基因的不同历史名称

### 1.2 消歧原则

1. **精确性**: 明确指代具体实体
2. **可追溯性**: 保留原始命名信息
3. **标准化**: 遵循权威命名规范
4. **兼容性**: 支持多数据库交叉引用

## 二、跨物种基因/蛋白消歧

### 2.1 命名规范对照

#### 人类基因（HGNC标准）
| 特征 | 规则 | 示例 |
|------|------|------|
| 基因符号 | 全大写 | TP53, BRCA1, EGFR |
| 蛋白名称 | 首字母大写 | p53蛋白, BRCA1蛋白 |
| 斜体书写 | 基因名用斜体 | *TP53* |

#### 小鼠基因（MGI标准）
| 特征 | 规则 | 示例 |
|------|------|------|
| 基因符号 | 首字母大写，其余小写 | Trp53, Brca1, Egfr |
| 蛋白名称 | 全大写 | TP53, BRCA1 |
| 斜体书写 | 基因名用斜体 | *Trp53* |

#### 模式生物命名
| 物种 | 基因符号 | 示例 |
|------|----------|------|
| 大鼠 | 全大写（或首字母大写） | Tp53 / TP53 |
| 果蝇 | 首字母大写 | p53, white |
| 线虫 | 小写+数字 | daf-2, age-1 |
| 酵母 | 全大写 | RAD53, SIR2 |
| 斑马鱼 | 小写 | tp53, brca1 |
| 拟南芥 | 全大写 | FLC, FT |
| 大肠杆菌 | 三个小写+大写 | lacZ, recA |

### 2.2 消歧标注格式

```
基础格式: 〖@基因名|物种:XXX〗
扩展格式: 〖@基因名|物种:XXX|数据库ID:XXX|符号:标准符号〗
```

## 三、基因家族消歧

### 3.1 p53家族

```

├── 人类
│   ├── 〖@TP53|家族:p53|功能:肿瘤抑制〗
│   ├── 〖@TP63|家族:p53|别名:p51/p73L〗
│   └── 〖@TP73|家族:p53|功能:发育调控〗
```

## 七、消歧决策流程

### 7.1 消歧检查清单

```
遇到实体名称时，按以下顺序检查:

1. 【物种识别】
   □ 是否明确提及物种？
   □ 上下文暗示的物种是什么？
   □ 实验材料来源物种？

2. 【基因/蛋白区分】
   □ 讨论的是基因还是蛋白？
   □ 是否需要同时标注基因和蛋白？

3. 【家族成员识别】
   □ 是否属于已知基因家族？
   □ 具体是家族中的哪个成员？

4. 【变体识别】
   □ 是否涉及剪接变体？
   □ 是否涉及突变体？
   □ 是否涉及亚型？

5. 【命名验证】
   □ 是否使用当前标准命名？
   □ 是否有别名需要标注？
   □ 是否有历史名称？

6. 【数据库交叉验证】
   □ HGNC/MGI/其他数据库验证
   □ UniProt蛋白验证
   □ NCBI Gene验证
```

## 九、消歧质量评估

### 9.1 评估指标

| 指标 | 目标值 | 计算方法 |
|------|--------|----------|
| 物种识别准确率 | ≥98% | 正确识别物种/总实体数 |
| 家族成员区分率 | ≥95% | 正确区分家族成员/家族实体数 |
| 变体标注完整率 | ≥90% | 完整标注变体/涉及变体实体数 |
| 数据库ID匹配率 | ≥95% | 成功匹配数据库ID/标注实体数 |

### 9.2 常见错误类型

| 错误类型 | 错误示例 | 正确标注 |
|----------|----------|----------|
| 物种混淆 | 〖@p53〗（小鼠实验却标人类） | 〖@Trp53|物种:小鼠〗 |
| 基因蛋白混淆 | 〖@p53蛋白〗 | 〖#p53蛋白〗 |
| 家族成员错误 | 〖@p63〗当作p53 | 〖@TP63|家族:p53〗 |
| 变体遗漏 | 〖@TP53〗（实际是Δ40变体） | 〖@TP53|变体:Δ40〗 |
| 历史名称未标注 | 〖@TP53〗（原文用p53） | 〖@TP53|别名:p53〗 |
