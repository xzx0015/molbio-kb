# SKILL 03a: 分子生物学实体标注 — 从原文到结构化实体的完整方法

> 为分子生物学课程内容设计12类实体标注体系，涵盖基因、蛋白、细胞结构到实验技术的全谱系生物学概念。

---

## 一、标注体系设计

### 1.1 核心原则

**"内容与样式分离"** — 用轻量Token标记原文，渲染交给程序。

Token标记的优势：
- 原文保持可读（标注后的markdown仍可直接阅读）
- 版本可控（git diff友好，纯文本差异清晰）
- 渲染无关（同一份标注文件可渲染为HTML、PDF等）
- 跨平台兼容（支持标准Markdown渲染器）

### 1.2 12类实体标记符号

| 符号 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `〖@X〗` | **基因** | 基因名称、基因符号、基因座 | `〖@lacZ〗`、`〖@p53〗` |
| `〖#X〗` | **蛋白** | 蛋白质、多肽、酶、蛋白复合物 | `〖#RNA聚合酶〗`、`〖#p53蛋白〗` |
| `〖$X〗` | **细胞结构** | 细胞器、亚细胞结构、细胞组分 | `〖$细胞核〗`、`〖$线粒体〗` |
| `〖%X〗` | **分子过程** | 生物学过程、分子反应、代谢途径 | `〖%转录〗`、`〖%翻译〗` |
| `〖&X〗` | **技术方法** | 实验技术、分析方法、操作步骤 | `〖&PCR〗`、`〖&Western Blot〗` |
| `〖*X〗` | **代谢物** | 小分子、代谢中间产物、信号分子 | `〖*ATP〗`、`〖*cAMP〗` |
| `〖=X〗` | **试剂材料** | 实验试剂、培养基、缓冲液、耗材 | `〖=琼脂糖〗`、`〖=LB培养基〗` |
| `〖!X〗` | **条件参数** | 温度、pH、浓度、时间等实验条件 | `〖!37°C〗`、`〖!pH7.4〗` |
| `〖>X〗` | **文献来源** | 经典文献、发现者、里程碑研究 | `〖>Watson&Crick1953〗` |
| `〖<X〗` | **核心概念** | 理论、法则、模型、假说 | `〖<中心法则〗` |
| `〖+X〗` | **疾病/表型** | 疾病、表型特征、生理状态 | `〖+癌症〗`、`〖+耐药性〗` |
| `〖:X〗` | **数值数据** | 定量数据、测量值、统计结果 | `〖:Tm=65°C〗` |

### 1.3 标注格式规范

#### 基础格式

```markdown
所有实体使用〖TYPE 内容〗格式，其中TYPE为上述12类符号之一。

示例：
- 〖@p53〗基因编码〖#p53蛋白〗，在〖+癌症〗发生中起关键作用。
- 〖%转录〗过程由〖#RNA聚合酶〗催化，发生在〖$细胞核〗中。
- 使用〖&PCR〗技术扩增〖@目的基因〗，反应条件为〖!95°C〗预变性。
```

#### 内联消歧语法

当同一实体在不同语境下指代不同概念时，使用消歧语法：

```markdown
格式：〖TYPE 显示名|规范名〗

示例：
- 〖@p53|人类p53〗与〖@p53|小鼠p53〗的序列相似性为80%
- 〖#P450|CYP3A4〗是药物代谢的主要酶
- 〖%复制|DNA复制〗与〖%复制|病毒复制〗机制不同
```

渲染输出：
```html
<span class="gene" title="类型：基因，规范名：人类p53" data-canonical="人类p53">p53</span>
```

#### 物种标注规范

基因和蛋白实体应明确标注物种来源：

```markdown
标准格式：
- 〖@基因名|物种〗 或 〖@基因名|物种:具体说明〗

示例：
- 〖@TP53|人类〗（人类p53基因）
- 〖@Trp53|小鼠〗（小鼠p53基因）
- 〖@lacZ|大肠杆菌〗（大肠杆菌lacZ基因）
- 〖@p53|人类:野生型〗（指定具体变体）
```

## 二、12类实体详细定义

### 2.1 基因 〖@X〗

**定义**：具有遗传功能的DNA片段，包括编码基因、非编码基因、调控序列等。

**标注范围**：
- ✓ 基因符号（如 p53, lacZ, GFP）
- ✓ 基因全称（如 tumor protein 53）
- ✓ 基因座名称（如 17p13.1）
- ✓ 基因变体（如 p53-R175H）

**不标注**：
- ✗ 普通DNA序列（如 "GC含量" 中的 GC）
- ✗ 泛指基因概念（如 "基因表达" 中的 "基因" 不单独标）

**示例**：
```markdown
- 〖@TP53〗基因突变与多种〖+癌症〗相关
- 〖@lacZ〗编码β-半乳糖苷酶
- 〖@EGFR〗是表皮生长因子受体基因
- 〖@BRCA1|人类〗突变增加乳腺癌风险
```

**边界情况**：
| 场景 | 处理方式 | 示例 |
|------|----------|------|
| 基因家族 | 标注具体成员 | 〖@p53家族〗（家族名）；〖@p63〗（具体成员） |
| 基因+蛋白同时提及 | 分别标注 | 〖@p53基因〗编码〖#p53蛋白〗 |
| 基因符号大小写 | 保留原文 | 〖@p53〗（人类）；〖@Trp53〗（小鼠） |

---

### 2.2 蛋白 〖#X〗

**定义**：由基因编码的氨基酸聚合物，包括单体蛋白、多亚基蛋白、酶、蛋白复合物。

**标注范围**：
- ✓ 蛋白名称（如 RNA聚合酶, p53蛋白）
- ✓ 酶名称（如 DNA聚合酶, 限制性内切酶）
- ✓ 蛋白复合物（如 核糖体, 蛋白酶体）
- ✓ 抗体（如 抗p53抗体）

**不标注**：
- ✗ 氨基酸残基（如 "丝氨酸磷酸化" 中的丝氨酸）
- ✗ 泛指蛋白质概念

**示例**：
```markdown
- 〖#RNA聚合酶〗催化RNA合成的〖%转录〗过程
- 〖#DNA聚合酶III〗是大肠杆菌主要的复制酶
- 〖#p53蛋白〗通过调控细胞周期发挥抑癌作用
- 〖#绿色荧光蛋白|GFP〗常用于细胞成像
```

**边界情况**：
| 场景 | 处理方式 | 示例 |
|------|----------|------|
| 酶 vs 蛋白 | 酶优先标为蛋白 | 〖#DNA聚合酶〗（酶也是蛋白） |
| 蛋白复合物 | 整体标注 | 〖#RNA聚合酶全酶〗 |
| 蛋白修饰状态 | 标注基础蛋白 | 〖#p53蛋白〗磷酸化 → 不标磷酸化状态 |

---

### 2.3 细胞结构 〖$X〗

**定义**：细胞的物理结构，包括细胞器、膜结构、细胞骨架等亚细胞组分。

**标注范围**：
- ✓ 细胞器（细胞核、线粒体、内质网、高尔基体等）
- ✓ 膜结构（细胞膜、核膜、线粒体内膜等）
- ✓ 细胞骨架（微管、微丝、中间纤维）
- ✓ 亚细胞区室（细胞质、核仁、核基质）

**不标注**：
- ✗ 整个细胞（如 "大肠杆菌细胞" 中的细胞）
- ✗ 组织/器官层次结构

**示例**：
```markdown
- DNA复制发生在〖$细胞核〗中
- 〖$线粒体〗是细胞的能量工厂
- 蛋白质在〖$内质网〗合成后经〖$高尔基体〗加工
- 〖$核孔复合物〗调控核质间物质运输
```

**边界情况**：
| 场景 | 处理方式 | 示例 |
|------|----------|------|
| 细胞器膜 | 标注具体膜结构 | 〖$线粒体内膜〗 |
| 细胞结构+功能 | 标注结构 | 〖$核糖体〗（结构）；不标"翻译场所" |
| 病毒结构 | 可标注 | 〖$病毒包膜〗 |

---

### 2.4 分子过程 〖%X〗

**定义**：生物体内的分子水平过程，包括代谢反应、基因表达过程、信号转导等。

**标注范围**：
- ✓ 中心法则过程（复制、转录、翻译、逆转录）
- ✓ 代谢途径（糖酵解、三羧酸循环、氧化磷酸化）
- ✓ 修饰过程（磷酸化、乙酰化、泛素化）
- ✓ 细胞过程（细胞周期、凋亡、分化）

**不标注**：
- ✗ 物理过程（如扩散、渗透）
- ✗ 实验操作（如离心、电泳）→ 标为技术方法

**示例**：
```markdown
- 遗传信息通过〖%转录〗和〖%翻译〗实现表达
- 〖%DNA复制〗是半保留复制过程
- 〖%糖酵解〗在细胞质中将葡萄糖分解为丙酮酸
- 〖%细胞凋亡〗是程序性细胞死亡过程
```

**边界情况**：
| 场景 | 处理方式 | 示例 |
|------|----------|------|
| 过程 vs 结果 | 标过程 | 〖%转录〗产生mRNA |
| 过程+位置 | 仅标过程 | 〖%翻译〗在核糖体上进行 → 不标核糖体 |
| 多步骤过程 | 可整体或分步 | 〖%基因表达〗或分别标〖%转录〗〖%翻译〗 |

---

### 2.5 技术方法 〖&X〗

**定义**：分子生物学研究中使用的实验技术、分析方法和操作步骤。

**标注范围**：
