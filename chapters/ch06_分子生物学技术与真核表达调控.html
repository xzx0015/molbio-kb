第6章分子生物学技术与真核表达调控

第六章：分子生物学技术

6.1 PCR技术

原理

1<中心法则告诉我们，DNA复制是遗传信息传递的基础。PCR（聚合酶链式反应）技术正是模拟这一过程，在体外快速扩增特定DNA片段。

2发明者：Kary Mullis 1983因此获得1993年诺贝尔化学奖。

反应体系

组分 作用 示例
模板DNA 扩增的原始材料 基因组DNA
引物 确定扩增起始位点 上下游各一条
DNA聚合酶 催化DNA合成 Taq酶
dNTPs 原料 dATP、dTTP、dGTP、dCTP
缓冲液 提供适宜环境 含Mg²⁺、pH 8.3

反应步骤

<span class="entity">94-95°C</span>变性 → <span class="entity">50-65°C</span>退火 → <span class="entity">72°C</span>延伸
     ↓
   30-35个循环
     ↓
  DNA指数扩增（2ⁿ）

3关键参数： - 变性温度：94-95°C，时间：30秒-2分钟 - 退火温度：取决于引物Tm值，通常低于Tm 5°C - 延伸时间：1分钟/kb

Taq酶特点

4Taq DNA聚合酶是从嗜热菌Thermus aquaticus中分离的： - ✅ 耐热，在95°C不失活 - ✅ 最适温度72-80°C - ❌ 无3'-5'外切酶活性，无校对功能 - ❌ 错误率较高（约10⁻⁴）

5高保真酶：如Pfu酶、Phusion酶，具有校对功能。

6.2 DNA测序技术

Sanger测序（第一代）

6原理：双脱氧终止法

DNA合成反应 + 少量双脱氧核苷酸（ddNTP）
         ↓
    随机终止产生不同长度片段
         ↓
    电泳分离，读取序列

7特点： - 读长：500-1000 bp - 准确率：>99.9% - 通量：低（每次反应一条序列）

高通量测序（第二代）

平台 技术原理 读长 应用
Illumina 桥式PCR + 荧光标记 150-300 bp 全基因组、RNA-seq
Ion Torrent 半导体测序 200-400 bp 靶向测序
SOLiD 连接测序 50-75 bp 重测序

8Illumina测序流程： 1. 文库制备：DNA片段化 + 接头连接 2. 簇生成：桥式PCR扩增 3. 测序：边合成边测序（SBS） 4. 数据分析：图像 → 碱基识别

第三代测序

9PacBio（单分子实时测序）： - 读长：10-20 kb，最长可达100 kb - 优势：直接检测DNA甲基化 - 应用：基因组组装、结构变异检测

10Oxford Nanopore： - 原理：纳米孔单分子测序 - 特点：便携、实时、超长读长 - 设备：MinION（USB大小）

限制性内切酶

酶 识别序列 末端

6.4 基因编辑技术

CRISPR-Cas9系统

12发现：Jennifer Doudna & Emmanuelle Charpentier 2012，获2020年诺贝尔化学奖。

13组成： - Cas9蛋白：核酸酶，切割DNA - sgRNA：向导RNA，引导Cas9到靶位点

14原理：

sgRNA识别靶序列（PAM: NGG）
        ↓
Cas9切割DNA双链
        ↓
细胞修复（NHEJ或HDR）
        ↓
基因敲除或精确编辑

15应用： - 遗传病治疗（如镰状细胞贫血） - 癌症免疫治疗 - 功能基因研究 - 作物改良

碱基编辑器

16CBE（胞嘧啶碱基编辑器）：C → T ABE（腺嘌呤碱基编辑器）：A → G

17优势：不切断DNA双链，减少随机插入/缺失

6.5 基因表达分析

RT-PCR

18原理：

RNA →<span class="entity">逆转录</span>→ cDNA →<span class="entity">PCR</span>→ 扩增产物

19关键酶：逆转录酶（如M-MLV、AMV）

20应用： - 检测基因表达水平 - 验证RNA-seq结果 - 病毒检测（如COVID-19RT-PCR检测）

RNA干扰（RNAi）

21机制： - dsRNA被Dicer切割为siRNA（21-23 nt） - siRNA与RISC复合物结合 - 靶向降解互补mRNA

22应用： - 基因功能研究（基因敲低） - 治疗（如年龄相关性黄斑变性）

本章小结

  • PCR是体外DNA扩增的核心技术
  • 测序技术从Sanger发展到高通量、单分子测序
  • 基因克隆依赖载体系统和限制酶
  • CRISPR-Cas9革命性地改变了基因编辑
  • RNA技术（RT-PCR、RNAi）是基因表达研究的重要工具

第六章：真核生物基因表达调控

6.1 真核基因调控概述

与原核生物的差异

特征 原核生物 真核生物
转录翻译 偶联 分隔（核膜）
基因组织 操纵子 单基因
染色质 无 有核小体
主要调控点 转录 多水平
RNA加工 无 复杂（加帽、加尾、剪接）

调控层次

23真核基因表达调控发生在多个层次：

染色质水平 → 转录水平 → RNA加工 → mRNA运输 → 翻译水平 → 蛋白质修饰
    ↑           ↑          ↑          ↑          ↑          ↑
表观遗传   转录因子   选择性剪接   核质运输   翻译起始   信号转导

6.2 染色质水平调控

DNA甲基化

24DNA甲基化： - 位点：CpG岛中的胞嘧啶（C→5mC） - 酶：DNA甲基转移酶（DNMT1维持，DNMT3a/3b从头） - 效应：通常抑制基因表达

25生物学意义： - 基因组印记（Imprinting）：亲本特异性表达 - X染色体失活：雌性X染色体剂量补偿 - 转座子沉默 - 肿瘤抑制基因沉默

组蛋白修饰

26组蛋白修饰：

修饰 位点 效应
乙酰化（Ac） 赖氨酸 激活转录
甲基化（Me） 赖氨酸/精氨酸 H3K4me3激活，H3K27me3抑制
磷酸化（P） 丝氨酸 调控染色质结构
泛素化（Ub） 赖氨酸 调控转录

27组蛋白密码假说： - 不同修饰组合产生特定调控效应 - 修饰之间相互影响（交叉对话）

染色质重塑

28染色质重塑复合物： - SWI/SNF：利用ATP水解能量重塑核小体 - ISWI、CHD、INO80家族

29功能： - 改变核小体位置 - 影响DNA可及性 - 调控转录因子结合

6.3 转录水平调控

顺式作用元件

30顺式作用元件：DNA上的调控序列

元件 位置 功能
启动子（Promoter） 转录起始点附近 基础转录装置组装
增强子（Enhancer） 远端（1kb-1Mb） 显著增强转录效率
沉默子（Silencer） 远端 抑制转录
绝缘子（Insulator） 边界 阻断增强子/沉默子作用

31增强子特征： - 方向无关性 - 位置灵活性（上游、下游、内含子） - 组织特异性 - 与启动子形成染色质环

反式作用因子

32反式作用因子：调控蛋白

33转录因子结构： - DNA结合域：识别特定序列 - 转录激活域：招募转录 machinery - 二聚化结构域：形成同源/异源二聚体 - 核定位信号（NLS）：进入细胞核

34常见DNA结合域： | 结构域 | 特征 | 代表因子 | |--------|------|---------| | 同源异型域（Homeodomain） | 螺旋-转角-螺旋变体 | HOX蛋白 | | 锌指（Zinc finger） | Zn²⁺稳定 | SP1、TFIIIA | | 碱性亮氨酸拉链（bZIP） | 亮氨酸拉链二聚化 | AP-1、CREB | | 螺旋-环-螺旋（HLH） | HLH二聚化 | MyoD、E47 |

转录调控网络

35组合调控： - 多个转录因子协同作用 - 产生特异性的时空表达模式 - 增强调控特异性

36实例： - MyoD与肌肉分化 - Oct4、Sox2、Nanog与干细胞多能性

6.4 转录后调控

选择性剪接

37选择性剪接： - 一个基因产生多种mRNA异构体 - 人类约95%的基因存在选择性剪接 - 增加蛋白质组多样性

38剪接模式： | 模式 | 说明 | |------|------| | 外显子跳跃 | 跳过某个外显子 | | 互斥外显子 | 两个外显子只保留一个 | | 内含子保留 | 保留某个内含子 | | 可变5'剪接位点 | 使用不同的5'剪接位点 | | 可变3'剪接位点 | 使用不同的3'剪接位点 |

39调控因子： - SR蛋白：促进剪接 - hnRNP蛋白</组织>：抑制剪接

RNA编辑

40<concept id="rna_editing">RNA编辑</concept>： - 转录后改变RNA序列 - 脱氨酶（ADAR、APOBEC）介导 - 实例：APOB mRNA的C→U编辑

RNA稳定性调控

41mRNA降解途径： 1. <entity type="概念">脱腺苷酸化：去除poly(A)尾 2. 脱帽：去除5'帽 3. 外切酶降解：5'→3'或3'→5'

42调控元件： - AU富含元件（ARE）：促进降解 - miRNA：通过RNA干扰抑制翻译或促进降解

miRNA调控

43miRNA（微小RNA）： - 长度：21-25 nt - 来源：RNA聚合酶II转录，经Drosha和Dicer加工 - 机制：与RISC复合物结合，靶向mRNA

44调控方式： - 完全互补：mRNA切割 - 部分互补：翻译抑制

6.5 翻译水平调控

全局翻译调控

45eIF2磷酸化： - 应激条件下eIF2α磷酸化 - 抑制全局翻译起始 - 但激活特定应激基因翻译

46mTOR通路： - mTOR激酶感知营养信号 - 磷酸化4E-BP，释放eIF4E - 促进翻译起始

特异性翻译调控

47铁响应元件（IRE）： - 铁调节蛋白（IRP）结合IRE - 铁缺乏时：抑制铁蛋白mRNA翻译 - 铁充足时：IRP失活，翻译进行

48上游开放阅读框（uORF）： - 位于主ORF上游 - 抑制主ORF翻译 - 应激条件下被绕过

6.6 信号转导与基因表达

信号通路

49MAPK通路： - 生长因子 → Ras → Raf → MEK → ERK - ERK入核激活转录因子

50Wnt/β-catenin通路： - Wnt信号稳定β-catenin - β-catenin入核激活TCF转录因子

51TGF-β/Smad通路： - TGF-β受体磷酸化Smad2/3 - Smad复合物入核调控转录

激素调控

52类固醇激素： - 脂溶性，可穿过细胞膜 - 与核受体结合 - 激素-受体复合物直接调控转录

53实例： - 雌激素受体（ER） - 糖皮质激素受体（GR）

本章小结

  • 真核基因调控发生在多水平，比原核更复杂
  • 染色质修饰（DNA甲基化、组蛋白修饰）是表观遗传调控的基础
  • 转录因子通过顺式元件和反式因子精细调控
  • 转录后调控（选择性剪接、miRNA）增加调控层次
  • 信号转导通路将胞外信号转化为基因表达变化
