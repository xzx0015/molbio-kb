# 第一章：绪论

## 1.1 分子生物学的定义

**分子生物学**（Molecular Biology）是研究生物大分子（核酸和蛋白质）的结构、功能及其相互作用，揭示生命活动分子机制的学科。

### 核心研究对象

| 类别 | 代表分子 | 功能 |
|------|---------|------|
| 核酸 | <entity type="分子">DNA</entity>、<entity type="分子">RNA</entity> | 遗传信息存储与传递 |
| 蛋白质 | <entity type="分子">酶</entity>、<entity type="分子">结构蛋白</entity> | 催化反应、结构支撑 |
| 复合物 | <entity type="复合物">核糖体</entity>、<entity type="复合物">染色体</entity> | 复杂功能执行 |

## 1.2 中心法则

<concept id="central_dogma">中心法则</concept>（Central Dogma）是分子生物学的核心理论框架，描述了遗传信息的流动方向：

```
DNA → RNA → 蛋白质
```

### 信息流动过程

1. **复制**（Replication）：<entity type="过程">DNA复制</entity> → DNA
2. **转录**（Transcription）：<entity type="过程">DNA转录</entity> → RNA  
3. **翻译**（Translation）：<entity type="过程">RNA翻译</entity> → 蛋白质

> **补充**：<entity type="过程">逆转录</entity>（Reverse Transcription）是中心法则的补充，RNA → DNA，由<entity type="酶">逆转录酶</entity>催化。

- 分子生物学研究生物大分子的结构与功能
