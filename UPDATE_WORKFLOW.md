## 📝 新条目创建规范

### 实体条目模板

```json
{
  "id": "新概念名称",
  "name": "中文名称",
  "nameEn": "English Name",
  "category": "类别",
  "firstReported": "YYYY",
  "importance": "core/major/basic",
  "definition": "简明定义",
  "mechanism": "作用机制",
  "applications": ["应用1", "应用2"],
  "relatedEntities": ["相关概念1", "相关概念2"],
  "keyPapers": [
    {
      "title": "论文标题",
      "authors": "作者",
      "journal": "期刊",
      "year": "YYYY",
      "doi": "DOI"
    }
  ],
  "updateLog": [
    { "date": "YYYY-MM-DD", "action": "创建条目" }
  ]
}
```

### 关系更新规范

```json
{
  "source": "新概念",
  "target": "已有概念",
  "type": "关系类型",
  "description": "关系描述",
  "evidence": "文献依据",
  "confidence": "high/medium/low"
}
```
