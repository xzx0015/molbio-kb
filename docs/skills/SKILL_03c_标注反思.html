系统性错误修正流程

一、反思流程概述

1.1 为什么需要反思？

实体标注是一个复杂的过程，容易出现系统性错误： - 一致性错误: 同一实体在不同位置标注不一致 - 分类错误: 实体被错误归类到不正确的类别 - 消歧错误: 跨物种/家族成员识别错误 - 格式错误: 标注格式不符合规范 - 遗漏错误: 重要实体未被标注

1.2 反思原则

  1. 系统性: 按检查清单全面审查
  2. 可追溯: 记录所有修正决策
  3. 迭代性: 多次审查直至达标
  4. 文档化: 错误案例入库避免重复

二、反思检查清单

2.1 一级检查：格式规范

□ 标注符号使用正确（@#$%&*=!><+:）
□ 标注括号使用〖〗而非[]或()
□ 属性分隔符使用|而非/
□ 属性名和值使用:连接
□ 无多余空格（<span class="entity">p53</span>而非〖 @p53 〗）

2.2 二级检查：分类准确性

□ @基因：是否为DNA序列/基因符号
□ #蛋白：是否为氨基酸序列/蛋白质
□ $细胞结构：是否为亚细胞定位/细胞器
□ %分子过程：是否为生物学过程/反应
□ &技术方法：是否为实验技术/方法
□ *代谢物：是否为代谢小分子
□ =试剂材料：是否为实验试剂/耗材
□ !条件参数：是否为实验条件/数值
□ >文献来源：是否为引用/参考文献
□ <核心概念：是否为理论/概念/原理
□ +疾病/表型：是否为疾病/表型描述
□ :数值数据：是否为定量测量值

2.3 三级检查：消歧准确性

□ 跨物种实体是否标注物种属性
□ 基因家族成员是否正确区分
□ 剪接变体是否标注变体信息
□ 蛋白亚型是否标注亚型编号
□ 历史名称是否标注别名/曾用名
□ 病毒基因是否标注来源病毒

2.4 四级检查：一致性

□ 同一实体全文标注格式一致
□ 同一实体全文分类一致
□ 缩写首次出现是否标注全称
□ 中英文实体是否统一规范
□ 数值单位是否统一格式

2.5 五级检查：完整性

□ 核心概念是否全部标注
□ 关键基因/蛋白是否全部标注
□ 重要实验技术是否全部标注
□ 关键文献引用是否标注
□ 数值数据是否全部标注
□ 疾病/表型是否全部标注

三、错误类型与修正策略

3.1 格式错误

错误类型 错误示例 修正示例 修正策略
符号错误 [@p53] p53 使用正确的全角括号
空格错误 〖 @p53 〗 p53 删除多余空格
分隔符错误 p53/物种:人类 p53 使用竖线分隔
属性格式错误 p53 p53 属性名:属性值
符号混用 p53蛋白 p53蛋白 基因用@，蛋白用#

3.2 分类错误

错误类型 错误示例 修正示例 判断标准
基因标为蛋白 TP53 TP53 基因符号用@
过程标为结构 转录 转录 过程用%
技术标为过程 PCR PCR 技术用&
代谢物标为试剂 ATP ATP 代谢物用*
概念标为文献 中心法则 中心法则 概念用<

3.3 消歧错误

错误类型 错误示例 修正示例 修正策略
物种遗漏 p53（小鼠实验） Trp53 添加上下文物种
家族混淆 p53（实际为p63） TP63 核实家族成员
变体遗漏 TP53（β变体） TP53 识别剪接变体
亚型遗漏 p53（γ亚型） p53 标注蛋白亚型
别名遗漏 TP53（原文用p53） TP53 保留原始名称

3.4 一致性错误

错误类型 错误示例 修正示例 修正策略
格式不一致 p53和TP53 统一为TP53 全文统一
属性不一致 p53和p53 统一添加物种 全文统一
缩写不一致 GFP和绿色荧光蛋白 统一为GFP 使用标准缩写
分类不一致 PCR和PCR 统一为PCR 技术用&

3.5 遗漏错误

遗漏类型 示例 修正示例 检查方法
基因遗漏 p53基因未标注 添加p53 基因符号扫描
蛋白遗漏 GFP未标注 添加GFP 蛋白名称扫描
技术遗漏 Western blot未标注 添加Western Blot 技术术语扫描
数值遗漏 37°C未标注 添加37°C 数值模式扫描
文献遗漏 Watson & Crick 1953未标注 添加Watson&Crick1953 引用格式扫描

4.1 自动化预检查

# 伪代码示例

def pre_check_annotation(text):
    errors = []

    # 1. 格式检查
    if has_halfwidth_brackets(text):
        errors.append("使用半角括号，应改为全角〖〗")

    # 2. 符号检查
    invalid_symbols = find_invalid_symbols(text)
    if invalid_symbols:
        errors.append(f"无效符号: {invalid_symbols}")

    # 3. 一致性检查
    inconsistent = check_consistency(text)
    if inconsistent:
        errors.append(f"不一致标注: {inconsistent}")

    # 4. 潜在遗漏
    potential_missing = scan_potential_entities(text)
    if potential_missing:
        errors.append(f"潜在遗漏: {potential_missing}")

    return errors

6.2 错误案例详情

案例1: p53基因蛋白混淆

原文: "p53蛋白在细胞核中发挥作用"
错误: <span class="entity">p53</span>在<span class="entity">细胞核</span>中发挥作用
修正: <span class="entity">p53蛋白</span>在<span class="entity">细胞核</span>中发挥作用
分析: 原文明确提到"蛋白"，应使用#而非@

案例2: 跨物种标注遗漏

原文: "小鼠p53基因敲除实验显示..."
错误: <span class="entity">p53</span>基因敲除实验显示...
修正: <span class="entity">Trp53</span>基因敲除实验显示...
分析: 实验材料为小鼠，必须标注物种

案例3: 过程与结构混淆

原文: "转录发生在细胞核中"
错误: <span class="entity">转录</span>发生在<span class="entity">细胞核</span>中
修正: <span class="entity">转录</span>发生在<span class="entity">细胞核</span>中
分析: 转录是生物学过程，用%而非$

案例4: 技术方法遗漏

原文: "通过PCR扩增目的片段"
错误: 通过PCR扩增目的片段
修正: 通过<span class="entity">PCR</span>扩增目的片段
分析: PCR是实验技术，必须标注

案例5: 数值数据遗漏

原文: "反应在37°C进行30分钟"
错误: 反应在37°C进行30分钟
修正: 反应在<span class="entity">37°C</span>进行<span class="entity">30分钟</span>
分析: 温度和时间都是条件参数，应标注
