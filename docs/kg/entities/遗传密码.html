遗传密码

Genetic Code | 第 6 章

核心概念 第 6 章

📖 定义

遗传密码是 mRNA 上核苷酸序列与蛋白质氨基酸序列之间的对应规则，由 64 个三联体密码子组成。

🔑 关键概念

密码子

mRNA 上连续的 3 个核苷酸，决定一个氨基酸或翻译信号

有义密码子

编码氨基酸的 61 个密码子

终止密码子

不编码氨基酸，导致翻译终止的 3 个密码子（UAA、UAG、UGA）

起始密码子

AUG，编码甲硫氨酸，标志翻译起始

简并性

多个密码子编码同一氨基酸的现象

摆动假说

Crick 提出的密码子-反密码子配对第三位的灵活性

📌 核心知识点

  • 遗传密码的特征：三联体、连续性、简并性、通用性、摆动性
  • 64 个密码子：61 个有义密码子 + 3 个终止密码子
  • AUG 是起始密码子兼甲硫氨酸密码子
  • 密码子简并性主要发生在第三位
  • 线粒体和某些生物使用变异的遗传密码
