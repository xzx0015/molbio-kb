第5章原核表达调控与基因组维持

第五章：原核生物基因表达调控

5.1 基因表达调控概述

调控层次

1基因表达调控发生在多个层次：

DNA → 转录 → RNA加工 → 翻译 → 蛋白质修饰
 ↑      ↑       ↑        ↑        ↑
染色质  转录   转录后   翻译    翻译后

2原核生物特点： - 转录和翻译偶联（无核膜分隔） - 以操纵子为单位组织基因 - 主要调控点在转录水平 - 快速响应环境变化

调控类型

类型 机制 实例
负调控 阻遏蛋白抑制转录 乳糖操纵子
正调控 激活蛋白促进转录 CAP-cAMP
双重调控 正负调控结合 乳糖操纵子

5.2 操纵子模型

操纵子结构

3操纵子（Operon）是原核生物基因调控的基本单位：

调节基因 - 启动子 - 操纵基因 - 结构基因群
              ↑         ↑
         RNA聚合酶   阻遏蛋白
元件 功能
调节基因（Regulatory gene） 编码调控蛋白（阻遏蛋白/激活蛋白）
启动子（Promoter） RNA聚合酶结合位点
操纵基因（Operator） 阻遏蛋白结合位点
结构基因（Structural genes） 编码功能相关的蛋白质

调控蛋白

4阻遏蛋白（Repressor）： - 结合操纵基因，阻止转录 - 可被诱导物失活（负调控） - 可被辅阻遏物激活（阻遏型操纵子）

5激活蛋白（Activator）： - 结合启动子附近，促进转录 - 通常需要效应物分子

5.3 乳糖操纵子

负调控机制

6乳糖操纵子（Lac operon）是负调控的经典模型：

7结构： - 调节基因lacI：编码阻遏蛋白 - 启动子P_lac：RNA聚合酶结合 - 操纵基因O：阻遏蛋白结合 - 结构基因：lacZ（β-半乳糖苷酶）、lacY（通透酶）、lacA（转乙酰基酶）

8调控过程：

条件 阻遏蛋白状态 操纵基因 转录
无乳糖 活性形式 结合 关闭
有乳糖 结合诱导物失活 不结合 开启

9诱导物： - 乳糖本身不是真正的诱导物 - 异乳糖（allolactose）是真正的诱导物 - 实验室常用IPTG（异丙基-β-D-硫代半乳糖苷）作为诱导物

正调控机制

10CAP蛋白（Catabolite Activator Protein）介导正调控：

11葡萄糖效应（分解代谢物阻遏）： - 葡萄糖存在时：优先利用葡萄糖，乳糖操纵子低表达 - 葡萄糖缺乏时：cAMP水平升高，激活乳糖操纵子

12机制：

葡萄糖缺乏 → cAMP升高 → cAMP-CAP复合物形成 → 结合启动子 → 转录激活

13CAP结合位点： - 位于启动子上游 - 结合后弯曲DNA，促进RNA聚合酶结合

综合调控

14乳糖操纵子的双重调控：

葡萄糖 乳糖 cAMP-CAP 阻遏蛋白 转录水平
+ - 不结合 结合 关闭
+ + 不结合 不结合 低水平
- - 结合 结合 关闭
- + 结合 不结合 高水平

15只有当葡萄糖缺乏（cAMP高）且有乳糖（阻遏蛋白失活）时，乳糖操纵子才高水平表达。

5.4 色氨酸操纵子

阻遏型调控

16色氨酸操纵子（Trp operon）是阻遏型操纵子：

17结构： - 调节基因trpR：编码阻遏蛋白 - 启动子、操纵基因 - 结构基因：trpE、trpD、trpC、trpB、trpA（色氨酸合成酶各亚基）

18调控过程：

条件 色氨酸作用 阻遏蛋白 转录
无色氨酸 无 无活性 开启
有色氨酸 辅阻遏物 有活性 关闭

衰减作用

19衰减作用（Attenuation）是色氨酸操纵子的精细调控机制：

20前导序列（Leader sequence）： - 位于操纵基因和结构基因之间 - 包含4个可互补配对的区域（1、2、3、4） - 编码14个氨基酸的前导肽（含2个Trp）

21衰减机制：

22情况1：色氨酸充足 - 前导肽顺利翻译 - 核糖体占据区域1，区域2自由 - 区域3-4配对形成终止子（发夹+poly-U） - 转录提前终止

23情况2：色氨酸缺乏 - 前导肽翻译停滞在Trp密码子（区域1） - 核糖体停滞，区域2自由 - 区域2-3配对（抗终止子） - 区域4无法配对，转录继续

24衰减效率： - 衰减作用可使转录降低约10倍 - 与阻遏作用协同，总调控可达700倍

5.5 其他调控机制

阿拉伯糖操纵子

25阿拉伯糖操纵子（Ara operon）：

26独特之处： - 同一蛋白（AraC）既是阻遏蛋白又是激活蛋白 - 阿拉伯糖作为效应物改变AraC构象

27调控模式： - 无阿拉伯糖：AraC作为阻遏蛋白 - 有阿拉伯糖：AraC作为激活蛋白，结合CAP-cAMP

SOS应答

28SOS应答：DNA损伤应激反应

29关键蛋白： - LexA：阻遏蛋白 - RecA：辅蛋白酶

30机制： 1. DNA损伤 → RecA激活 2. RecA促进LexA自切割 3. LexA失活，SOS基因表达 4. DNA修复后，LexA重新积累

31SOS基因： - 修复基因（uvrA、uvrB等） - 易错聚合酶（Pol IV、Pol V） - 细胞分裂抑制（sulA）

热休克应答

32σ³²因子： - 识别热休克基因启动子 - 温度升高时表达 - 保护蛋白质免受热损伤

5.6 翻译水平调控

核糖体蛋白合成

33自体调控（Autogenous regulation）： - 核糖体蛋白结合自身mRNA的SD序列 - 抑制自身翻译 - 协调核糖体蛋白与rRNA的合成

反义RNA调控

34机制： - 反义RNA与mRNA互补配对 - 阻止核糖体结合或促进降解 - 调控质粒拷贝数等

本章小结

  • 原核生物基因表达以操纵子为单位组织
  • 乳糖操纵子是负调控和正调控的经典模型
  • 色氨酸操纵子通过阻遏和衰减双重机制精细调控
  • 同一调控蛋白可在不同条件下发挥不同作用
  • 翻译水平调控是转录调控的补充

第五章：基因组维持

5.1 DNA损伤

损伤类型

35DNA损伤分为两大类：

36自发性损伤： | 类型 | 机制 | 频率 | |------|------|------| | 脱嘌呤 | 糖苷键水解 | 10⁴/细胞/天 | | 脱氨基 | 胞嘧啶→尿嘧啶 | 100/细胞/天 | | 氧化损伤 | 活性氧（ROS） | 10³/细胞/天 |

37诱发性损伤： | 诱变剂 | 损伤类型 | |--------|---------| | UV | 嘧啶二聚体（CPD、6-4光产物） | | 电离辐射 | 双链断裂、碱基氧化 | | 烷化剂 | 碱基烷基化 | | 嵌入剂 | 碱基插入/缺失 |

常见DNA损伤

38嘧啶二聚体： - UV诱导，相邻嘧啶形成环丁烷环 - 阻碍DNA聚合酶前进 - 导致复制停滞

39氧化损伤： - 8-oxoG（8-氧鸟嘌呤）：G→T突变 - 胸腺嘧啶乙二醇 - 由活性氧（ROS）产生

5.2 DNA修复机制

直接修复

40直接修复：直接逆转损伤

类型 酶 机制
光修复 光裂合酶 光依赖，裂解嘧啶二聚体
O⁶-甲基鸟嘌呤修复 MGMT 转移甲基到自身
单链断裂修复 DNA连接酶 直接连接

切除修复

41切除修复：切除损伤片段，重新合成

42碱基切除修复（BER）：

损伤碱基 → 糖基化酶切除 → AP内切酶切割 → DNA聚合酶填补 → DNA连接酶连接
  • DNA糖基化酶：识别并切除损伤碱基
  • AP位点：无碱基位点
  • AP内切酶：切割AP位点

43核苷酸切除修复（NER）：

损伤识别 → 解旋 → 切割（两侧） → 切除寡核苷酸 → 填补 → 连接
  • 修复大的螺旋扭曲损伤（如嘧啶二聚体）
  • TFIIH复合物：解旋
  • XPG、XPF-ERCC1：切割

44全基因组NER vs 转录偶联NER： - GG-NER：全基因组扫描 - TC-NER：优先修复活跃转录基因

错配修复

45错配修复（MMR）：

46识别： - MutS：识别错配 - MutL：介导 - MutH：切割（原核）

47过程： - 识别新合成链（甲基化标记或缺口） - 切除包含错配的片段 - 重新合成

48临床意义： - 林奇综合征（遗传性非息肉性结直肠癌） - MMR基因突变导致微卫星不稳定

双链断裂修复

49双链断裂修复：最危险的DNA损伤

50同源重组修复（HRR）： - 需要同源模板（姐妹染色单体） - 高保真修复 - 主要发生在S/G2期

51关键蛋白： - RAD51：真核生物RecA同源物 - BRCA1/BRCA2：招募RAD51

52非同源末端连接（NHEJ）： - 直接连接断裂末端 - 不需要同源模板 - 可能产生突变（末端处理） - 主要发生在G1期

53关键蛋白： - DNA-PKcs：识别双链断裂 - Ku70/Ku80：结合DNA末端 - DNA连接酶IV：连接

5.3 同源重组

重组机制

54同源重组：同源序列间的DNA交换

55Holliday模型： 1. 双链断裂（或单链缺口） 2. 链侵入：单链侵入同源双链 3. D-loop形成 4. Holliday连接体形成 5. 分支迁移 6. 拆分（切开方式决定产物）

重组的生物学功能

功能 说明
DNA修复 修复双链断裂
遗传多样性 减数分裂中交换
复制重启 复制叉崩溃后恢复
端粒维持 替代端粒延长（ALT）

减数分裂重组

56过程： - 联会：同源染色体配对 - 联会复合体形成 - Spo11：诱导双链断裂 - 重组：形成交叉（Chiasma）

57结果： - 遗传物质交换 - 确保染色体正确分离

5.4 转座子

转座子类型

58转座子（Transposon）：可移动的遗传元件

类型 机制 代表
DNA转座子 "剪切-粘贴" P元件（果蝇）
逆转录转座子 "复制-粘贴" LINE、SINE

59逆转录转座子： - LINE-1（L1）：长散在重复序列，占人类基因组17% - Alu：短散在重复序列，占人类基因组11%

转座机制

60DNA转座子： 1. 转座酶识别末端反向重复（IR） 2. 切除转座子 3. 插入新位点 4. 靶位点重复（TSD）

61逆转录转座子： 1. 转录为RNA 2. 逆转录为cDNA 3. 整合入基因组

转座子的影响

62有害： - 插入基因内部导致突变 - 引起染色体断裂

63有益： - 基因组进化动力 - 调控元件来源 - 遗传多样性

5.5 端粒与端粒酶

端粒结构

64端粒（Telomere）：染色体末端的保护结构

  • 序列：人类为TTAGGG重复（5-15 kb）
  • 结构：T环（T-loop），3'端回折
  • 功能：防止染色体末端被识别为DNA损伤

端粒酶

65端粒酶（Telomerase）： - 核糖核蛋白复合物 - RNA组分（TERC）：模板序列 - 蛋白质组分（TERT）：逆转录酶活性

66机制：

端粒DNA 3'端 → 端粒酶结合 → 以RNA为模板延伸 → 移位 → 重复

端粒与衰老

67端粒缩短： - 正常体细胞：端粒酶活性低，端粒逐代缩短 - Hayflick极限：细胞分裂约50-60次后衰老

68端粒酶与癌症： - 85-90%的癌细胞端粒酶重新激活 - 维持端粒长度，获得无限增殖能力 - 端粒酶抑制剂是潜在的抗癌药物靶点

本章小结

  • DNA损伤分为自发性和诱发性，类型多样
  • 修复机制包括直接修复、切除修复、错配修复、双链断裂修复
  • 同源重组是修复双链断裂的高保真方式
  • 转座子是可移动遗传元件，对基因组进化有重要意义
  • 端粒保护染色体末端，端粒酶维持端粒长度
