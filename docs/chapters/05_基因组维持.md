# 第五章：基因组维持

## 5.1 DNA损伤

### 损伤类型

<concept id="dna_damage">DNA损伤</concept>分为两大类：

**自发性损伤**：
| 类型 | 机制 | 频率 |
|------|------|------|
| <entity type="损伤">脱嘌呤</entity> | 糖苷键水解 | 10⁴/细胞/天 |
| <entity type="损伤">脱氨基</entity> | 胞嘧啶→尿嘧啶 | 100/细胞/天 |
| <entity type="损伤">氧化损伤</entity> | 活性氧（ROS） | 10³/细胞/天 |

**诱发性损伤**：
| 诱变剂 | 损伤类型 |
|--------|---------|
| <entity type="诱变剂">UV</entity> | 嘧啶二聚体（CPD、6-4光产物） |
| <entity type="诱变剂">电离辐射</entity> | 双链断裂、碱基氧化 |
| <entity type="诱变剂">烷化剂</entity> | 碱基烷基化 |
| <entity type="诱变剂">嵌入剂</entity> | 碱基插入/缺失 |

### 常见DNA损伤

**嘧啶二聚体**：
- UV诱导，相邻嘧啶形成环丁烷环
- 阻碍DNA聚合酶前进
- 导致复制停滞

**氧化损伤**：
- <entity type="损伤">8-oxoG</entity>（8-氧鸟嘌呤）：G→T突变
- <entity type="损伤">胸腺嘧啶乙二醇</entity>
- 由<entity type="分子">活性氧</entity>（ROS）产生

## 5.2 DNA修复机制

### 直接修复

<concept id="direct_repair">直接修复</concept>：直接逆转损伤

| 类型 | 酶 | 机制 |
|------|-----|------|
| <entity type="修复">光修复</entity> | <entity type="酶">光裂合酶</entity> | 光依赖，裂解嘧啶二聚体 |
| <entity type="修复">O⁶-甲基鸟嘌呤修复</entity> | <entity type="酶">MGMT</entity> | 转移甲基到自身 |
| <entity type="修复">单链断裂修复</entity> | <entity type="酶">DNA连接酶</entity> | 直接连接 |

### 切除修复

<concept id="excision_repair">切除修复</concept>：切除损伤片段，重新合成

**碱基切除修复**（<entity type="修复">BER</entity>）：
```
损伤碱基 → 糖基化酶切除 → AP内切酶切割 → DNA聚合酶填补 → DNA连接酶连接
```

- <entity type="酶">DNA糖基化酶</entity>：识别并切除损伤碱基
- <entity type="概念">AP位点</entity>：无碱基位点
- <entity type="酶">AP内切酶</entity>：切割AP位点

**核苷酸切除修复**（<entity type="修复">NER</entity>）：
```
损伤识别 → 解旋 → 切割（两侧） → 切除寡核苷酸 → 填补 → 连接
```

- 修复大的螺旋扭曲损伤（如嘧啶二聚体）
- <entity type="复合物">TFIIH</entity>复合物：解旋
- <entity type="酶">XPG</entity>、<entity type="酶">XPF-ERCC1</entity>：切割

**全基因组NER vs 转录偶联NER**：
- <entity type="修复">GG-NER</entity>：全基因组扫描
- <entity type="修复">TC-NER</entity>：优先修复活跃转录基因

### 错配修复

<entity type="修复">错配修复</entity>（<entity type="修复">MMR</entity>）：

**识别**：
- <entity type="酶">MutS</entity>：识别错配
- <entity type="酶">MutL</entity>：介导
- <entity type="酶">MutH</entity>：切割（原核）

**过程**：
- 识别新合成链（甲基化标记或缺口）
- 切除包含错配的片段
- 重新合成

**临床意义**：
- <entity type="疾病">林奇综合征</entity>（遗传性非息肉性结直肠癌）
- MMR基因突变导致微卫星不稳定

### 双链断裂修复

<concept id="dsb_repair">双链断裂修复</concept>：最危险的DNA损伤

**同源重组修复**（<entity type="修复">HRR</entity>）：
- 需要同源模板（姐妹染色单体）
- 高保真修复
- 主要发生在S/G2期

**关键蛋白**：
- <entity type="蛋白">RAD51</entity>：真核生物<entity type="蛋白">RecA</entity>同源物
- <entity type="蛋白">BRCA1</entity>/<entity type="蛋白">BRCA2</entity>：招募RAD51

**非同源末端连接**（<entity type="修复">NHEJ</entity>）：
- 直接连接断裂末端
- 不需要同源模板
- 可能产生突变（末端处理）
- 主要发生在G1期

**关键蛋白**：
- <entity type="激酶">DNA-PKcs</entity>：识别双链断裂
- <entity type="蛋白">Ku70</entity>/<entity type="蛋白">Ku80</entity>：结合DNA末端
- <entity type="酶">DNA连接酶IV</entity>：连接

## 5.3 同源重组

### 重组机制

<concept id="homologous_recombination">同源重组</concept>：同源序列间的DNA交换

**Holliday模型**：
1. 双链断裂（或单链缺口）
2. <entity type="过程">链侵入</entity>：单链侵入同源双链
3. <entity type="结构">D-loop</entity>形成
4. <entity type="结构">Holliday连接体</entity>形成
5. 分支迁移
6. 拆分（切开方式决定产物）

### 重组的生物学功能

| 功能 | 说明 |
|------|------|
| DNA修复 | 修复双链断裂 |
| 遗传多样性 | 减数分裂中交换 |
| 复制重启 | 复制叉崩溃后恢复 |
| 端粒维持 | 替代端粒延长（ALT） |

### 减数分裂重组

**过程**：
- <entity type="过程">联会</entity>：同源染色体配对
- <entity type="结构">联会复合体</entity>形成
- <entity type="酶">Spo11</entity>：诱导双链断裂
- 重组：形成<entity type="结构">交叉</entity>（Chiasma）

**结果**：
- 遗传物质交换
- 确保染色体正确分离

## 5.4 转座子

### 转座子类型

<concept id="transposon">转座子</concept>（Transposon）：可移动的遗传元件

| 类型 | 机制 | 代表 |
|------|------|------|
| <entity type="转座子">DNA转座子</entity> | "剪切-粘贴" | <entity type="转座子">P元件</entity>（果蝇） |
| <entity type="转座子">逆转录转座子</entity> | "复制-粘贴" | <entity type="转座子">LINE</entity>、<entity type="转座子">SINE</entity> |

**逆转录转座子**：
- <entity type="转座子">LINE-1</entity>（L1）：长散在重复序列，占人类基因组17%
- <entity type="转座子">Alu</entity>：短散在重复序列，占人类基因组11%

### 转座机制

**DNA转座子**：
1. 转座酶识别末端反向重复（IR）
2. 切除转座子
3. 插入新位点
4. 靶位点重复（TSD）

**逆转录转座子**：
1. 转录为RNA
2. 逆转录为cDNA
3. 整合入基因组

### 转座子的影响

**有害**：
- 插入基因内部导致突变
- 引起染色体断裂

**有益**：
- 基因组进化动力
- 调控元件来源
- 遗传多样性

## 5.5 端粒与端粒酶

### 端粒结构

<entity type="结构">端粒</entity>（Telomere）：染色体末端的保护结构

- 序列：人类为<entity type="序列">TTAGGG</entity>重复（5-15 kb）
- 结构：<entity type="结构">T环</entity>（T-loop），3'端回折
- 功能：防止染色体末端被识别为DNA损伤

### 端粒酶

<entity type="酶">端粒酶</entity>（Telomerase）：
- <entity type="复合物">核糖核蛋白</entity>复合物
- <entity type="分子">RNA组分</entity>（<entity type="RNA">TERC</entity>）：模板序列
- <entity type="蛋白">蛋白质组分</entity>（<entity type="蛋白">TERT</entity>）：逆转录酶活性

**机制**：
```
端粒DNA 3'端 → 端粒酶结合 → 以RNA为模板延伸 → 移位 → 重复
```

### 端粒与衰老

**端粒缩短**：
- 正常体细胞：端粒酶活性低，端粒逐代缩短
- <entity type="概念">Hayflick极限</entity>：细胞分裂约50-60次后衰老

**端粒酶与癌症**：
- 85-90%的癌细胞端粒酶重新激活
- 维持端粒长度，获得无限增殖能力
- 端粒酶抑制剂是潜在的抗癌药物靶点

## 本章小结

- DNA损伤分为自发性和诱发性，类型多样
- 修复机制包括直接修复、切除修复、错配修复、双链断裂修复
- 同源重组是修复双链断裂的高保真方式
- 转座子是可移动遗传元件，对基因组进化有重要意义
- 端粒保护染色体末端，端粒酶维持端粒长度
