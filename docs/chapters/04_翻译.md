# 第四章：翻译

## 4.1 遗传密码

### 密码子特征

<concept id="genetic_code">遗传密码</concept>：mRNA上编码氨基酸的三联体密码子

| 特征 | 说明 |
|------|------|
| 密码子数 | 64个（4³） |
| 编码氨基酸 | 61个 |
| 终止密码子 | 3个：UAA、UAG、UGA |
| 起始密码子 | AUG（编码Met） |

**密码子特点**：
- **简并性**：多个密码子编码同一氨基酸（如Leu有6个密码子）
- **通用性**：几乎所有生物共用（线粒体有例外）
- **连续性**：无标点，不重叠
- **方向性**：5'→3'阅读
- **不重叠性**：每个核苷酸只属于一个密码子

|--------|---|---|---|---|
| **U** | Phe | Ser | Tyr | Cys |

| | Leu | Ser | STOP | Trp |

| | Leu | Pro | Gln | Arg |

## 4.2 tRNA与氨基酸活化

### tRNA结构

<entity type="分子">tRNA</entity>（转运RNA）的功能：
- 携带特定氨基酸
- 识别mRNA上的密码子（<entity type="概念">反密码子</entity>）

**结构特征**：
- 长度：73-93个核苷酸
- 三叶草二级结构
- L形三级结构
- <entity type="概念">氨基酸臂</entity>：3'端CCA，结合氨基酸
- <entity type="概念">反密码子环</entity>：识别密码子
- <entity type="概念">D环</entity>和<entity type="概念">TψC环</entity>：结构稳定

### 氨基酸活化

<entity type="酶">氨酰-tRNA合成酶</entity>（aaRS）：
- 催化氨基酸与tRNA连接
- 高度特异性（一种酶对应一种氨基酸）
- 两步反应：
  1. 氨基酸 + ATP → 氨酰-AMP + PPi
  2. 氨酰-AMP + tRNA → 氨酰-tRNA + AMP

**校对机制**：
- 合成酶的编辑活性
- 识别tRNA的反密码子和受体茎
- 确保翻译保真性

### 摆动配对

<concept id="wobble">摆动假说</concept>：反密码子5'位碱基与密码子3'位配对有一定灵活性

| 反密码子5'位 | 可配对密码子3'位 |
|-------------|-----------------|
| C | G |
| A | U |
| U | A或G |
| G | C或U |
| I（次黄嘌呤） | U、C或A |

## 4.3 核糖体

### 核糖体结构

<entity type="复合物">核糖体</entity>是蛋白质合成的场所：

| 亚基 | 原核 | 真核 |
|------|------|------|
| 大亚基 | 50S（23S+5S rRNA + 蛋白质） | 60S（28S+5.8S+5S rRNA + 蛋白质） |
| 小亚基 | 30S（16S rRNA + 蛋白质） | 40S（18S rRNA + 蛋白质） |
| 完整 | 70S | 80S |

**rRNA功能**：
- 催化肽键形成（核酶活性）
- 提供结构框架
- 参与tRNA和mRNA结合

### 核糖体功能位点

| 位点 | 名称 | 功能 |
|------|------|------|
| **A位** | 氨酰位（Aminoacyl site） | 接受新氨酰-tRNA |
| **P位** | 肽酰位（Peptidyl site） | 携带正在延伸的肽链 |
| **E位** | 出口位（Exit site） | 释放空载tRNA |

### 核糖体组装

**原核生物**：
- 30S亚基：16S rRNA + 21种蛋白质
- 50S亚基：23S rRNA + 5S rRNA + 34种蛋白质

**真核生物**：
- 在核仁中组装
- 需要snoRNA指导rRNA修饰
- 组装后输出到细胞质

## 4.4 翻译过程

### 起始阶段

**原核生物**：
1. **小亚基识别mRNA**：<entity type="概念">SD序列</entity>（Shine-Dalgarno）与16S rRNA 3'端互补
2. **起始tRNA进入**：fMet-tRNAᶠᴹᵉᵗ进入P位
3. **大亚基结合**：形成70S起始复合物

**真核生物**：
1. **5'帽识别**：eIF4F复合物识别m⁷G帽
2. **扫描机制**：43S前起始复合物从5'端扫描到AUG
3. **起始密码子识别**：Kozak序列（GCCRCC**AUG**G）
4. **大亚基结合**：形成80S起始复合物

**起始因子**：
| 因子 | 功能 |
|------|------|
| IF1/IF2/IF3 | 原核起始因子 |
| eIF1-eIF6 | 真核起始因子（eIF4F识别帽结构） |

### 延伸阶段

**延伸循环**：
1. **氨酰-tRNA进入A位**：EF-Tu（原核）/eEF1A（真核）护送
2. **肽键形成**：<entity type="酶">肽酰转移酶</entity>催化（rRNA的核酶活性）
3. **核糖体移位**：EF-G（原核）/eEF2（真核）驱动
4. **空载tRNA释放**：从E位离开

**延伸因子**：
| 原核 | 真核 | 功能 |
|------|------|------|
| EF-Tu | eEF1A | 护送氨酰-tRNA进入A位 |
| EF-Ts | eEF1B | 再生EF-Tu/eEF1A |
| EF-G | eEF2 | 驱动核糖体移位 |

### 终止阶段

**终止信号**：
- 终止密码子：UAA、UAG、UGA
- 无对应的氨酰-tRNA

**释放因子**：
| 因子 | 识别密码子 |
|------|-----------|
| RF1（原核） | UAA、UAG |
| RF2（原核） | UAA、UGA |
| RF3（原核） | 促进RF1/RF2释放 |
| eRF1（真核） | 所有终止密码子 |

**终止过程**：
1. 释放因子识别终止密码子
2. 肽酰转移酶活性改变：水解肽酰-tRNA
3. 多肽链释放
4. 核糖体解离

## 4.6 翻译调控

### 翻译起始调控

**eIF4E结合蛋白**（<entity type="蛋白">4E-BP</entity>）：
- 与<entity type="因子">eIF4E</entity>竞争结合
- <entity type="激酶">mTOR激酶</entity>磷酸化4E-BP，释放eIF4E
- 营养信号调控翻译

**铁响应元件**（<entity type="概念">IRE</entity>）：
- <entity type="蛋白">铁调节蛋白</entity>（IRP）结合IRE
- 铁缺乏时：抑制<entity type="蛋白">铁蛋白</entity>mRNA翻译
- 铁充足时：IRP失活，翻译进行

### 全局调控

**eIF2磷酸化**：
- 应激条件下eIF2α磷酸化
- 抑制全局翻译起始
- 但激活特定应激基因翻译

**miRNA调控**：
- <entity type="分子">miRNA</entity>与<entity type="复合物">RISC复合物</entity>结合
- 靶向mRNA抑制翻译或促进降解

## 本章小结

- 遗传密码具有简并性、通用性、连续性
- tRNA通过反密码子识别密码子，携带特定氨基酸
- 核糖体是蛋白质合成的场所，具有A、P、E三个位点
- 翻译分起始、延伸、终止三个阶段
- 翻译后修饰决定蛋白质的最终功能和定位
- 翻译水平调控是基因表达调控的重要环节
