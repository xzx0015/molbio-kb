第一章：绪论

1.1 分子生物学的定义

1分子生物学（Molecular Biology）是研究生物大分子（核酸和蛋白质）的结构、功能及其相互作用，揭示生命活动分子机制的学科。

核心研究对象

类别 代表分子 功能
核酸 DNA、RNA 遗传信息存储与传递
蛋白质 酶、结构蛋白 催化反应、结构支撑
复合物 核糖体、染色体 复杂功能执行

1.2 中心法则

2中心法则（Central Dogma）是分子生物学的核心理论框架，描述了遗传信息的流动方向：

DNA → RNA → 蛋白质

信息流动过程

  1. 复制（Replication）：DNA复制 → DNA
  2. 转录（Transcription）：DNA转录 → RNA
  3. 翻译（Translation）：RNA翻译 → 蛋白质

3补充：逆转录（Reverse Transcription）是中心法则的补充，RNA → DNA，由逆转录酶催化。

1.3 分子生物学发展简史

里程碑事件

年份 事件 意义
1953 Watson & Crick 提出DNA双螺旋模型 分子生物学诞生
1961 Jacob & Monod 提出操纵子模型 基因调控理论建立
1972 重组DNA技术诞生 基因工程时代开启
1990 人类基因组计划启动 大规模测序时代
2012 CRISPR-Cas9基因编辑技术 精准基因编辑时代

1.4 分子生物学研究方法

核心技术

  • 核酸提取：从细胞中分离DNA/RNA
  • PCR（聚合酶链式反应）：扩增特定DNA片段
  • DNA测序：测定DNA碱基序列
  • 基因克隆：将基因插入载体进行扩增
  • 基因编辑：精确修改基因序列

本章小结

  • 分子生物学研究生物大分子的结构与功能
  • 中心法则描述了遗传信息 DNA → RNA → 蛋白质的流动
  • 技术发展推动了分子生物学的快速进步
