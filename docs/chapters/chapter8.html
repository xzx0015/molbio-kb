8.1 基因组学

DNA 测序技术

代际技术特点
第一代Sanger 测序读长 500-1000 bp，准确率高
第二代Illumina高通量，短读长（150-300 bp）
第三代PacBio/Nanopore长读长（10-20 kb），实时测序

全基因组分析

  • WGS（全基因组测序）：检测 SNV、Indel、SV、CNV
  • WES（全外显子测序）：只测外显子区域，成本较低

8.2 转录组学

RNA-seq 技术

应用：

  • 基因表达定量
  • 可变剪接分析
  • 新转录本发现
  • 融合基因检测

单细胞转录组

scRNA-seq 技术：

  • 10x Genomics：高通量，3'端捕获
  • Smart-seq2：全长覆盖
  • Drop-seq：低成本

空间转录组

保留组织空间信息，结合组织学和转录组学

8.3 蛋白质组学

质谱技术

LC-MS/MS：液相色谱分离 + 质谱检测

定量方法

方法原理
SILAC稳定同位素标记
iTRAQ/TMT同位素标记
Label-free无标记，基于肽段强度

8.4 表观基因组学

ChIP-seq

染色质免疫沉淀 + 测序，鉴定蛋白质结合位点

DNA 甲基化测序

  • WGBS：全基因组亚硫酸氢盐测序，金标准
  • RRBS：简化代表性亚硫酸氢盐测序，成本较低

染色质构象

  • Hi-C：检测染色质三维结构
  • Capture-C：高分辨率检测特定区域互作

8.5 前沿技术

基因治疗

递送系统类型
病毒载体AAV、慢病毒、腺病毒
非病毒载体脂质纳米颗粒（LNP）
基因编辑CRISPR 体内递送

合成生物学

  • 基因线路：逻辑门、振荡器、开关
  • 细胞工厂：微生物合成药物、生物燃料

人工智能与生物学

  • AlphaFold：蛋白质结构预测
  • 深度学习：基因表达预测、药物发现

本章小结

  • 组学技术从基因组、转录组、蛋白质组、代谢组多层次研究生物系统
  • 单细胞和空间技术揭示细胞异质性和组织微环境
  • 表观基因组学研究基因调控的表观遗传机制
  • 前沿技术（基因治疗、合成生物学、AI）正在改变生物医学研究
