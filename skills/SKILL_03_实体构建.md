# SKILL 03: 实体构建 — 分子生物学知识库实体体系总览

> 从基因、蛋白到实验技术，构建分子生物学领域的结构化实体标注体系。

---

## 一、实体构建阶段概述

### 1.1 阶段定位

实体构建是知识库建设的核心阶段，承接内容采集与结构分析，为后续关系构建和本体构建提供基础实体数据。

```
内容采集(SKILL_01) → 结构分析(SKILL_02) → 【实体构建(SKILL_03)】 → 关系构建(SKILL_04) → 本体构建(SKILL_05)
```

### 1.2 阶段目标

1. **建立实体类型体系** — 定义12类生物学实体及其标注规范
2. **完成实体标注** — 对课程内容进行标准化实体标记
3. **实现实体消歧** — 解决同名实体跨物种/跨语境的歧义问题
4. **持续质量反思** — 建立标注错误修正的系统性流程

## 二、12类实体体系概览

### 2.1 实体类型总表

| 符号 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `〖@X〗` | 基因 | 基因名称、基因符号 | `〖@lacZ〗`、`〖@p53〗` |
| `〖#X〗` | 蛋白 | 蛋白质、酶、复合物 | `〖#RNA聚合酶〗` |
| `〖$X〗` | 细胞结构 | 细胞器、亚细胞结构 | `〖$细胞核〗` |
| `〖%X〗` | 分子过程 | 生物学过程、反应 | `〖%转录〗`、`〖%翻译〗` |
| `〖&X〗` | 技术方法 | 实验技术、分析方法 | `〖&PCR〗`、`〖&Western Blot〗` |
| `〖*X〗` | 代谢物 | 小分子、代谢中间产物 | `〖*ATP〗`、`〖*NADH〗` |
| `〖=X〗` | 试剂材料 | 实验试剂、培养基、耗材 | `〖=琼脂糖〗` |
| `〖!X〗` | 条件参数 | 温度、pH、浓度等参数 | `〖!37°C〗`、`〖!pH7.4〗` |
| `〖>X〗` | 文献来源 | 经典文献、发现者 | `〖>Watson&Crick1953〗` |
| `〖<X〗` | 核心概念 | 理论、法则、模型 | `〖<中心法则〗` |
| `〖+X〗` | 疾病/表型 | 疾病、表型特征 | `〖+癌症〗`、`〖+耐药性〗` |
| `〖:X〗` | 数值数据 | 定量数据、测量值 | `〖:Tm=65°C〗` |

### 2.2 实体分类原则

#### 原则一：功能优先于位置

同一实体按其主要功能分类，而非物理位置：
- `〖@p53〗`（基因）→ 不标为 `〖#p53蛋白〗`
- `〖#RNA聚合酶〗`（蛋白）→ 不标为 `〖%转录〗`的过程

#### 原则二：具体实例优先于抽象概念

- `〖@lacZ〗`（具体基因）vs `〖<乳糖操纵子模型〗`（概念）
- `〖&PCR〗`（具体技术）vs `〖<PCR原理〗`（概念）

#### 原则三：标注粒度适中

- ✓ `〖@p53基因〗` 或 `〖@p53〗`（上下文明确时）
- ✗ ~~`〖@p53基因〗蛋白产物`~~（不跨词标注）

## 三、标注流程

### 3.1 四阶段流水线

```
原始课程内容(.md)
    ↓ 阶段1: AI语义标注
标注文件(chapter_md/*.tagged.md)
    ↓ 阶段2: 质量控制
验证通过的标注文件
    ↓ 阶段3: 实体提取与索引构建
entity_index.json + entity_aliases.json
    ↓ 阶段4: 消歧与增强
disambiguation_map.json + species_map.json
```

### 3.2 阶段详解

#### 阶段1：AI语义标注

使用LLM对课程内容进行批量实体标注，详见 `SKILL_03a_实体标注.md`。

#### 阶段2：质量控制

- **语法检查**：验证标记闭合、格式正确
- **完整性验证**：确保标注不改变原文
- **人工抽检**：每批次抽检2-3章

#### 阶段3：实体提取

从标注文件中提取实体，构建索引：
- 规范实体名（别名合并后）
- 出现位置（章节、段落）
- 引用频次统计

#### 阶段4：消歧增强

- 同名基因/蛋白跨物种消歧
- 别名映射（如 p53 ↔ TP53 ↔ 肿瘤蛋白53）
- 物种标注（人类/小鼠/大肠杆菌等）

## 四、与古籍标注体系的对比

### 4.1 继承的设计模式

从《史记》实体标注体系（SKILL_03a）继承的核心模式：

| 设计模式 | 古籍应用 | 分子生物学应用 |
|----------|----------|----------------|
| Token标记 | `〖@刘邦〗` | `〖@p53〗` |
| 内联消歧 | `〖@台\|吕台〗` | `〖@p53\|人类〗` |
| 元数据外挂 | disambiguation_map.json | species_map.json |
| 别名系统 | entity_aliases.json | gene_aliases.json |

### 4.2 生物学特有扩展

| 扩展点 | 说明 |
|--------|------|
| 物种维度 | 基因/蛋白必须标注物种来源 |
| 数值类型 | 实验参数、定量数据需要结构化 |
| 层级关系 | 基因→转录本→蛋白的层级 |
| 动态过程 | 分子过程的时序性 |

## 六、质量指标

### 6.1 标注覆盖率

- 目标：核心概念覆盖率 ≥ 95%
- 方法：抽样检查关键章节

### 6.2 消歧准确率

- 目标：同名实体消歧准确率 ≥ 90%
- 方法：人工验证歧义case

### 6.3 标注一致性

- 目标：同一实体标注一致率 ≥ 98%
- 方法：跨章节一致性检查
